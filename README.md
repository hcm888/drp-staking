# DRP质押赎回DApp

专为TP钱包优化的质押赎回DApp，可一键部署到Vercel。

## 🚀 功能特性

- ✅ **TP钱包专版优化** - 完美适配TP钱包移动端
- ✅ **质押赎回管理** - 查看质押余额、赎回操作
- ✅ **实时收益显示** - 展示待领取收益
- ✅ **质押记录查看** - 查看历史质押记录
- ✅ **交易确认流程** - 完整的交易确认和状态显示
- ✅ **安全连接** - 支持TP钱包、MetaMask等Web3钱包
- ✅ **响应式设计** - 适配各种移动设备屏幕
- ✅ **Vercel部署** - 一键部署到Vercel平台

## 📱 在TP钱包中使用

### 方法一：直接访问Vercel部署地址
1. 打开TP钱包
2. 切换到DApp浏览器
3. 输入部署后的Vercel地址：`https://your-app-name.vercel.app`

### 方法二：扫描二维码访问
1. 将部署后的地址生成二维码
2. 在TP钱包中扫描二维码访问
